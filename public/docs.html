API Documentation

Available Endpoints

  • /api/ytsearch

    GET - Search for YouTube videos by query.

    Usage: `/api/ytsearch?query=example`

    Response: JSON list of video results with title, URL, views, etc.

  • /api/download/audio

    GET - Download audio from YouTube videos.

    Usage: `/api/download/audio?url=https://youtube.com/watch?v=example&quality=128`

    Response: JSON with download URL.

  • /api/download/video

    GET - Download video from YouTube.

    Usage: `/api/download/video?url=https://youtube.com/watch?v=example&quality=360`

    Response: JSON with download URL.
